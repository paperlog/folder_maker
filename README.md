# folder_maker



大学の講義資料や課題を整理するためのフォルダーを一括生成するツールです。
deadline.py(deadline.exe)は提出期限の管理を手助けするためのものです。ダウンロードしなくともcreate_university_folder.py(create_university_folder.exe)の動作に影響はありません。



[Demonstration Video](https://youtu.be/S05aLCXM3R8)



## 1. 環境

- pythonの実行環境が必須。



## 2. 動作説明

実行ファイルと同じ階層に、ルートフォルダー（デフォルト：`example`）を作成します。



- 既に `example` フォルダーが存在する場合、その中に新しい講義フォルダーが追加されます。

- 既存のフォルダーやファイルが削除されることはありません。



> [!TIP]

> **入力を楽にする方法** > 大学の履修登録画面から授業名の範囲をコピーし、AI（Geminiなど）に「授業名を抜き出して改行して出力してください」と依頼すると、入力用のリストを簡単に作成できます。



## 3. 入出力イメージ



### 入力例

```text

授業A

授業B

授業C

end
```

### 出力例（フォルダー構成）

```

example

 |- 授業A

 | |- 授業

 | |- 提出済み

 | |- 未提出

 |- 授業B

 | |- 授業

 | |- 提出済み

 | |- 未提出

 |- deadline.py (または deadline.exe)
```

---

## 4. 提出期限確認ツール (deadline.py)

作成された各講義フォルダー内の「未提出」フォルダーにある課題の期限を一覧表示します。



### 使い方

各講義の 未提出 フォルダー内に、課題ごとのフォルダー（例：レポート第1回,class1,file1）を作成します。



そのフォルダー内に deadline.txt を作成し、1行目に期限を記述します。



deadline.py を実行すると、全講義の期限をまとめて表示します。



### 構成ルール

授業名 ＞ 未提出 ＞ 課題名フォルダ ＞ deadline.txt (1行目に期限を記述)

※「未提出」直下に置いたテキストファイルは読み込まれません。
